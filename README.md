# mememax-kit
mememax临时刷tx

活动地址: https://mememax.com/mypage

## 使用
1. 创建.env文件
2. 填写SENDER_PRIVATE_KEY和RECIPIENT_ADDRESS

```
SENDER_PRIVATE_KEY=
RECIPIENT_ADDRESS=
```

3. 运行`pnpm start`
4. 查看输出

```
Transaction successful! URL: https://memecorescan.io/tx/0x1234567890abcdef1234567890abcdef1234567890abcdef1234567890abcdef
```
